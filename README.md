## Grafisk Teknik DT200G | Projektuppgift

En webbutik för en fiktiv Experimental Drum & Bass musikartist som önskar sälja sina låtar online. 
Webbplatsen innehåller grundläggande logik för varukorgen. Låtar kan läggas till eller tas bort från varukorgen och 'köpas'. Antalet produkter räknas och visas i 'varukorg' knappen och alla produkter sparas temporärt i sessionStorage
så att informationen om dem kan delas mellan undersidor.
